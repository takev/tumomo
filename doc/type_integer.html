Taula - Documentation - Integer

by Take Vos (2013)

An integer is a simple type, meaning that it is not like a class and can not be inherinted from. However if a single short integer is used as an attribute for the class then an instance of the class may be passed as an integer on the CPU.

Short integer

A short integer is an integer which will fit in a CPU register. These days that is often 64-bit.

Long integer

Longer integers up to 2**32 number of bits are supported by LLVM and by Taula. It is an error if a integer is inferred to become larger.
